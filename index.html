                      # PLC 原理及应用

                      -PLC（可编程逻辑控制器）是一种专门用于工业控制领域的数字运算装置，它利用可编程存储器、逻辑操作、序列控制、定时、计数 器与算术运算等功能，对输入的各种信息进行处理，形成控制输出来实现各 种自动化控制任务。

                      # 供电技术

                      -供用电技术主要研究电力学、电机学、供用电工程以及电力系统施工、维护、自动控制、技术开发等方面的基本知识和技能，在电力 行业一线进行电力系统供用电各环节的技术开发和管理等。例如：输电、变电线路及设备的安装调试，高、低压输电线路的维护，变电站用电设备的使用与检修等。

                      # 电力拖动自动控制系统

                      -“电力拖动自动控制系统”课程旨在介绍电力拖动控制系统的原理、分析方法和设计方法等内容，是一门帮助学习者掌握电力拖动控制系统基本理论、实现设计与实践的课程。

                      # DSP 原理及应用

                      -DSP 是一种专门用于数字信号处理的微处理器，具有高性能、低功耗和高度可编程的特点。 它的工作原理包括指令获取、解码、数据处理、结果存储和控 制流程等阶段。 DSP 在通信、音频、视频、雷达、医疗等领域有着广泛的应用。

                      # 计算机控制技术

                      -计算机控制技术》是自动化、自动控制、电气工程及其自动化等电气信息类专业的一门专业主干课程。该课程主要包括计算机控制系统的特点、组成、分类以及计算机 控制系统的发展趋势，计算机控制系统的基本输入输出接口与过程通道，数字控制技术，数字 PID 控制算法，常规及复杂控制技术，应用程序设计与实现技术，分布式测控网络技术，计算机控制系统设计与实现。

                      # 可编程逻辑器件及应用

                      -《可编程器件及应用》是实践性强的技术性基础课程。 以大规模可编程逻辑器件为设计载体，以硬件描述语言为系统逻辑描述的主要表达方式，以计算机、大规模可编程逻辑器件的开 发软件及实验开发系统为设计工具，通过相关开发软件，自动完成用软件方式设计的电子系统到硬件系统的逻辑编译、逻辑化简、逻辑分割、逻辑综合及优化、逻辑布局布线、逻辑仿真、直 至对于特定目标芯片的适配编译、逻辑映射、编程下载等工作，最终形成集成电子系统或专用集成芯片的一门新技术，或称 IES/ASIC 自动设计技术。
